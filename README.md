# Product Banner Component

## :clipboard: Опис завдання
Реалізовано дві кастомні секції Shopify для маркетингових банерів та відображення вибраних товарів із підтримкою сортування
Секції:
- Promo Banners — адаптивна промо-сітка з банерами.
- Featured Products — інтерактивний список товарів із динамічним перезавантаженням.

## :white_check_mark: Реалізований функціонал

### Promo Banners
- [x] Заголовок та підзаголовок з налаштувань секції
- [x] Вирівнювання тексту (left/center/right)
- [x] Кольорова схема: світла, темна, акцентна
- [x] Відступи секції (0–100px)
- [x] Динамічні блоки banner

### Функціональність банерів
- [x] Зображення через image_picker
- [x] Заголовок банера та кнопка
- [x] Посилання на будь-яку URL
- [x] Підтримка aspect ratio (16:9, 1:1, 3:2)
- [x] Адаптивна сітка

### Featured Products
Основні компоненти
- [x] Заголовок секції
- [x] Вибір колекції через settings
- [x] Обмеження кількості товарів (2–12)
- [x] Опційне відображення ціни
- [x] Опційне відображення рейтингу
- [x] Опційна кнопка «Додати в кошик»

### Сортування
- [x] Зbest-selling
- [x] price-low
- [x] price-high
- [x] title-ascending

### AJAX та інтерактивність
- Оновлення секції без перезавантаженн
- Перемикання сортування через <select>
- AJAX-додавання в кошик через fetch('/cart/add.js')
- Оновлення міні-кошика після додавання
